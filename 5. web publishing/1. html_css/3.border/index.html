테두리를 정의하는 다른 방법

border-syle에서 실선인지 점선인지 정함
border-color에서 색을 정함
border-width에서 선의 굵기를 정함
잘 선호하지는 않음
border-top:3px dotted blue;
border-bottom:4px dashed red;
이런식으로 각각 지정할 수 있다.

어떤 요소(ex input)들은 default로 border가 설정되어 있다.
그럴때는 border:none; 이나 border:0;을 사용하면 border를 제거할 수 있다.
background-clip : border-box; background-clip : padding-box; background-clip : content-box; -> 칠해지는 영역을 조절할 수 있다.
