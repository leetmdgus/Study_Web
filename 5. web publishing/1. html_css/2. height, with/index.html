글이 border를 넘는경우가 발생한다.

overflow:visible 은 기본 값이다.
overflow:hiden을 사용하면 border보다 넘는 글은 안보이게 할 수 있다.
혹은 scroll을 사용하는 방법도 있다.
비슷한 값으로는 auto가 있다.
scroll은 overflow가 발생하지 않아도 scroll을 띄우는데
auto는 내용이 많지 않으면 scroll을 띄우지 않음
